Скрипт Validate_script.py выполняет следующие функции:
- Перезаписывает лог-файл
- Ищет в окружени каталог event с данными формата json
- Ищет в окружени каталог schema со схемами json.
- Проверяет схемы json на соответствие мета-схеме Draft4Validator.META_SCHEMA модуля jsonschema
- Проверяет файлы в каталоге event на соответствие каждой схеме
- Создает лог-файл и записывает результаты проверки.

Использование:
1. Поместите файл Validate_script.py в каталог с данными event и schema
2. Запустите скрипт
